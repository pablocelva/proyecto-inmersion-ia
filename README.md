# Proyecto Inmersion IA

Este es un proyecto de inmersión en inteligencia artificial. Aquí se documentarán los objetivos, metodología y resultados del proyecto.

## Objetivos
- Aprender las bases de la IA
- Implementar modelos de aprendizaje automático
- Evaluar el rendimiento de los modelos

## Metodología
1. Revisión bibliográfica
2. Implementación práctica
3. Evaluación y análisis de resultados

## Resultados
Los resultados se documentarán a medida que se avance en el proyecto.
